## 🎞️ 项目介绍

> UI 使用react开发，UI库使用antd。[项目地址](https://github.com/liumengniu/audio-editor). \
> 音频剪辑使用 ffmpeg 开发。 \
> 波形图基于 wavesurfer.js开发 。 \
> 由于浏览器对于文件操作的限制以及浏览器运行内存大小的限制，纯web端实现不如客户端实现 。 \

## 🎨 运行指南

> 项目运行需要node环境 ，推荐node14+，作者使用的是node 16.13.1

> 1. 安装依赖

```bash
   npm install
```

> 2. 运行项目

```bash
   npm run start
```

> 3. 编译项目

```bash
   npm run build
```
